# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-03-24

### Added

- AI chat with streaming responses powered by the pi coding agent SDK
- Multi-session support with independent conversations per project
- File browser with Monaco editor for viewing and editing project files
- Git integration — view diffs, stage/unstage files, commit, push, and create PRs
- Embedded terminal via xterm.js
- Plan mode for deliberate, read-only planning workflows
- Model selection with thinking level controls
- Web fetch tool for referencing live documentation during conversations
- Skills system for loading domain-specific instructions
- Browser preview panel for local development servers
- Sidebar with project management and session grouping
- Cross-platform support (macOS, Windows, Linux)
